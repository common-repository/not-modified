Prevents post modified dates from being changed when you update your posts.

== Description ==

This plugin prevents post modified dates from being changed when you update your posts. Just activate the plugin, if want your post modified dates to stay the same while you doing the changes.

== Installation ==

= Automatic Install =

1. Log into your WordPress dashboard and go to Plugins &rarr; Add New
2. Search for "Not Modified"
3. Click "Install Now" under the Not Modified plugin
4. Click "Activate Now"

= Manual Install =

1. Download the plugin from the download button on this page
2. Unzip the file, and upload the resulting `rudr-not-modified` folder to your `/wp-content/plugins` directory
3. Log into your WordPress dashboard and go to Plugins
4. Click "Activate" under the Not Modified plugin
